The idea of a boat comes afloat

The choosing of a project

When we came together our first plan was to try and use as many of the instruments that we had already learned about from other projects, these were the 3D printer and CNC laser cutting machine. The newest machine we had access to this time was a machine called shopbot a CNC cutting machine that can cut in 2.5D but that is when the z axes has depth control. This allows us to still only cut in the xy plane, but in different layers of the xy planes we can cut differently. Our idea was to try to use the full potential of the equipment and so we set out to find something that would require the need of using the full potential of the 2.5D shopbot. Lastly if at all possible we would like to implement simple motor functions or automation that uses basic electronics.

The first and last idea we got was to create a motorized boat, first iterations were boats that were driven by paddles similar as seen here it this image: This could have been possible but we did not settle with this idea. The next idea was very similar, but in stead of paddles it would work like the boats that sailed the Mississippi river in the cowboy era, like this: These first ideas were good but not great, they lacked the WOW FACTOR. After a few days of thinking we checked out some other boat ideas and stumbled upon a category of boats that were mainly 3D printed, with some electronics as an exception, this sparked our interest. We browsed through multiple pages of these kind of boats, most notably Thingiverse. Then the idea was formed, make a jet propelled boat, other details will be sorted later.
