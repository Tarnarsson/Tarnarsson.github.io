Finding an icon for this website

Icon

Adding an icon to the top right corner of the tab, to make this happen I began by finding a nice cute image to use as a test subject. I made a quick google search for "Fox icons" and enden up with this cutie:

I started out trying to use this stack overflow article. A quick summary: It tells you to add the icons to a place where you can use them, I chose to create a floder called icons that stores the photos i will use as icons. Then write a bit of code into the head block of all your html files. This works but I did not get full coverage of web browsers icons with this method. For example my Safari web browser did not get an icon as can be seen on the photo bellow.

So next I tried out Favicons that creates icons and code for them, you simply upload a photo and the website returns to you a file that you put into your website along with a few lines of code and where to put them.
