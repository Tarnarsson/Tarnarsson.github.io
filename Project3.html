Watering can

Made with 3D printing

In the last years 3D printing has been growing rapidly, it is a very material efficient way to create objects because it is built from the ground up wrather than taking a solid block and milling away material. Although some builds require some structure materials that is nothing compared to what is created with the other techniques. This project focuses on creating an object that can not be created with methods that remove material.

Finding a suitable project

Fortunately I had already thought of what I was going to create. In the process of looking for something to create in project 2 I had stumbled upon a watering can that looks like this.

Now with this image as a guide I went and found out how to model this in Fusion 360. The process started with setting up some guidelines that resembeled the shape of the watering can that I wanted. Not having alot of modeling expirience at this time showed through here I had never really used features such as sweep or loft. So to compesate against my lack of knowledge I simplified the design significantly. And the body became a straight cone instead of this shape that looks like a rooster with its chest in the air. So in the next three pictures the process of modeling from start to finish

Here I have set up the main guidelines that will become the handle, body and nozzle of the watering can.

Then I create a solid body and somehow managed to make it hollow.

And then in the end there were only finishing touches left, like making holes to put water in on the top and angleing the end of the nozzle. The final step is to export the file as a mesh (stl file) so the printer can read it later.

The printing

The newly created stl file is the put into an applicaiton known as prusa slicer, the application creates g code for the printer. The printer used in this course is the Prusa i3 MK3S+ and the appliction is specially created to work with all prusa printers.

The 3d modeling will be of the final project when it has been finished
